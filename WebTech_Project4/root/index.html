Welcome to Version Control & SEO Demo Site

Introduction to Version Control

This website demonstrates the fundamentals of version control using Git and GitHub. The site is designed to meet accessibility and SEO best practices.

Students will learn how to clone repositories, commit changes, and push updates using terminal commands integrated with Visual Studio Code.

Enhanced Index Page Customization

The index page includes SEO meta tags, images with alt attributes, and CSS transitions for navigation links. It also features a structured layout with headings, paragraphs, and a list.

Accessibility is a key focus: images are optimized to 250x250 pixels, and colour contrast has been verified using the WebAIM Contrast Checker.

  • SEO Meta Tags: Title, Description, Author
  • Images: 3 images, each 250x250 pixels with alt text
  • Navigation: CSS transitions with external links
  • Content Structure: 1 H1, ≥2 H2s, ≥4 paragraphs, and a list
